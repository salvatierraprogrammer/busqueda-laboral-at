## 📦 Instalación local

Clonar el repositorio:

```bash
git clone https://github.com/salvatierraprogrammer/busqueda-laboral-at.git
```

Instalar dependencias:

```bash
npm install
```

Ejecutar en desarrollo:

```bash
npm run dev
```

Este proyecto nace con la idea de mejorar el acceso al trabajo para acompañantes terapéuticos, centralizando oportunidades laborales y recursos en una sola plataforma.
